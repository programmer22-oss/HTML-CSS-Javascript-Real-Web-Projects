# Responsive Calculator Web Component

This project implements a simple, responsive calculator as a single-page web component using HTML, CSS, and JavaScript.

## Features

*   Basic arithmetic operations (+, -, *, /)
*   Clear (C) and All Clear (AC) buttons
*   Responsive design adapts to different screen sizes
*   Error handling for invalid expressions

## Usage

1.  Include the HTML, CSS, and JavaScript code in your project.
2.  The calculator will automatically render on the page.

## Technologies Used

*   HTML
*   CSS
*   JavaScript
